# 자바_연산자

### 논리 연산자
 - 두개의 논리값을 연산해주는 연산자
    - && (and)       
    - ll (or)
    - ==
    - !=


### 증감 연산자
 - ++: 변수에 담긴 값을 1 증가
 - --: 변수에 담긴 값을 1 감소
    -  a++ , a--

### 산술 연산자
 - +-*/ %

### 대입 연산자
 - **=**
    - a=b
    
### 복합 대입 연산자
 - 산술 연산자와 대입 연산자가 결합되어있는 형태
    - a+=3; a-=3;

### 비교연산자/관계연산자 (이항연산자)
 - 두값을 비교하는 연산자
   - <  > <= >=

### 삼항 연산자
 - 항목 3개를 가지고 연산해주는 연산자.

    ```java 
    String check=(a%2==0)?"짝수":"홀수";
    ```
